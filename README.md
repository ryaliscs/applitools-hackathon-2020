# Applitools Cross Browser Testing Hackathon Instructions to execute

## Programming Language
```
Java - 1.8
```
## Build tool
```
Gradle
Plugin used: Eclipse Buildship
```

## Frame works used:
```
Junit 
eyes-selenium-java3
```

## Project Structure
```
|--applitools-hackathon-2020
	|--ModernTestsV1
	|--ModernTestsV2
```

## How to run tests
### Executing ModernTestsV1 tests
```
cd ./ModernTestsV1
gradlew build
gradlew test
```

### Executing ModernTestsV2 tests
```
cd ./ModernTestsV1
gradlew build
gradlew test
```
